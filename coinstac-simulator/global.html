    createUserData(dirGlobopt, csvopt, delimiteropt) → {array}

    Takes a input directory glob and/or a csv and returns an array of users constructed from its structure. The glob data files may be delimited text files or json arrays, an optional delimiter can be passed in for text files.

    The file structure should be as follows:

    Glob:

    path/to/Data Dir/ ├── some-user1 │ └── x │ │ ├── some-data.txt │ │ └── some-data.txt │ └── y │ ├── some-data.txt │ └── some-data.txt └── some-user2 └── x │ ├── some-data.txt │ └── some-data.txt └── y ├── some-data.txt └── some-data.txt

    CSV:

    user,t,u,v some-user1,1,2,2 some-user2,4,5,6
